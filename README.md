# LyricsXR-iOS

- [x] Dynamic island
- [x] Live activity
- [x] Widget
- [ ] Widget's Background
- [ ] Widget's next line preview

## Requirements

- iOS 14.0

## Clone

```
git clone --recurse-submodules -j8 git@github.com:bexonpak/LyricsXR-iOS.git
```

### Build

- Xcode 12
- macOS 11
